# Rename File

## Estrutura do Projeto

O projeto contém os seguintes arquivos e diretórios:

- `.vs`: Possivelmente um diretório de configuração do Visual Studio.
- `bin/Debug/net8.0-windows`: Provavelmente contém os artefatos de compilação do projeto para a plataforma Windows.
- `obj`: Possivelmente contém arquivos de objeto gerados durante o processo de compilação.
- `Form1.Designer.cs`: Arquivo de código-fonte que define o design do formulário 1.
- `Form1.cs`: Arquivo de código-fonte que contém a lógica por trás do formulário 1.
- `Form1.resx`: Arquivo de recursos associado ao formulário 1.
- `Program.cs`: Arquivo de código-fonte que contém o método `Main()` da aplicação.
- `RenameFile.csproj`: Arquivo de projeto do Visual Studio.
- `RenameFile.csproj.user`: Arquivo de configuração do usuário do projeto.
- `RenameFile.sln`: Arquivo de solução do Visual Studio.
